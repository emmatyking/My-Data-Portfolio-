📊 Women's Clothing E-Commerce Review Analysis

Data tells a story, and this dashboard uncovers valuable insights from customer reviews in the women's fashion industry.

Using Excel, I analyzed customer ratings, age distribution, department performance, and clothing categories to identify patterns in customer preferences and shopping behavior.

Key insights explored:
✅ Rating distribution across clothing classes
✅ Customer age segmentation by department
✅ Top-performing clothing IDs based on ratings
✅ Department and division-level performance analysis
✅ Product category engagement trends

This project demonstrates how data analytics can transform raw customer feedback into actionable business insights that support better decision-making, improve customer satisfaction, and drive product strategy.

Tools Used: Microsoft Excel | Pivot Tables | Charts | Dashboard Design | Data Visualization
